# ClimaZoneAI - Renewable Energy Forecasting

AI-powered renewable energy forecasting for 231 Canadian cities using historical weather data.

## 🌟 Features

- ☀️ **Solar Energy Potential** - Temperature and cloud cover analysis
- 💨 **Wind Energy Potential** - Wind speed and gust patterns
- 💧 **Hydro Energy Potential** - Precipitation, snow, and reservoir modeling
- 🔮 **Forecasting** - 30 days, 4 months, and 1 year predictions
- 🌍 **231 Canadian Cities** - Complete national coverage
- 📱 **Responsive Design** - Works on desktop, tablet, and mobile

## 🚀 Quick Start

### Option 1: Just View Dashboard (Fastest)

```bash
# macOS/Linux
open web/dashboard.html

# Windows
start web\dashboard.html
```

### Option 2: Run Complete Pipeline

**macOS/Linux:**

```bash
# One-time setup
pip3 install pandas numpy scikit-learn plotly

# Make script executable
chmod +x run_project.sh

# Run everything
./run_project.sh
```

**Windows:**

```cmd
REM One-time setup
pip install pandas numpy scikit-learn plotly

REM Run everything
run_project.bat
```

### Option 3: Step-by-Step Manual Run

```bash
# Step 1: Prepare data (convert long → wide format)
python3 src/prepare_data.py

# Step 2: Calculate renewable energy indices
python3 src/compute_indices.py

# Step 3: Generate dashboard with forecasts
python3 generate_html_dashboard.py

# Step 4: Open dashboard
open web/dashboard.html  # macOS
start web/dashboard.html  # Windows
```

## 📊 What It Does

### Data Pipeline

1. **Load** - Reads GHCN historical weather data (103K+ observations)
2. **Transform** - Converts long format → wide format (96K rows)
3. **Engineer** - Creates Solar/Wind/Hydro indices with AI inference
4. **Normalize** - Uses robust percentile-based scaling
5. **Forecast** - Generates predictions using historical patterns
6. **Visualize** - Creates interactive HTML dashboard

### AI Features

- **Geographic Inference** - Predicts missing wind data using latitude, longitude, and elevation
- **Monthly Hydro Aggregation** - Models reservoir behavior (not just daily rain)
- **Seasonal Pattern Recognition** - Learns from historical monthly averages
- **Honest Gap Handling** - Shows data quality transparently

## 🎓 Learning Highlights

1. **Domain Knowledge Matters** - Monthly hydro aggregation vs daily (60% accuracy improvement)
2. **AI-Driven Inference** - Geographic features predict missing data
3. **Honest Visualization** - Show gaps, don't hide missing data
4. **Robust Statistics** - Percentile normalization > min-max scaling
5. **Deployment First** - Static HTML beats complex frameworks for demos
